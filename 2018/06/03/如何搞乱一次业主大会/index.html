如何搞乱一次业主大会

如何搞乱一次业主大会

引子

  1. 小区品质差，开发商遗留了大量工程质量问题；
  2. 物业不作为，包括在服务的专业性，解决遗留问题上的态度，财务费用的不透明方面。从我的角度来看，就是垃圾物业的典型了。
  3. 由于开发商工程中偷工减料，出现一住户家里面污水倒灌，在居委会、业委会、物管、开发商、住户等的诡异协调下，进而演化出一出未经业主同意动用大修基金的事情，最后变成了责任不明确，处理不清晰透明的事情。
  4. 前段时期，业委会贴出更换物管的相关通告。启动更换物业的事宜。
  5. 鉴于此，物管随后开始挨家挨户摸底，做思想工作，灌输思想。

业主大会

  1. 今天是2018.6.3日，业委会牵头更换物管，开启动牵头大会；本来是件简单的事情，出现了围攻业委会成员的事情。现场一片混乱。
  2. 主要围绕业委会随意动用业主大修基金。一群人开始围攻，看起来更细是事前安排好的。本来是物管动用了大修基金，被硬生生的安在了业委会的头上。改变了舆论导向。大会是开不下去了。现场乱作一团。中心围着的是业委会的人，围着业委会的是那些强势的“业主”，再是看热闹各自讨论的业主，互相表达的彼此的看法。最外面的是保安和物业的那班人，还有那些围在他们周边的人，每个人都在不停的表达，各怀鬼胎，各自装着各自的心思。
  3. 看到那些称赞物业好的人气的青筋暴起，简直可笑。看着那些受害的业主，声嘶力竭。可又有什么办法呢，都有人拍桌子，站在桌子上了，一看就不是什么善茬。哎，孤立无援的业委会，看着好可伶，可是又有多少人能帮他们说句话呢，在外围看了很久，也思索了一下，觉得真是可怕。
  4. 哎，可悲。
  5. 哎，可聊。
  6. 想要一个好小区，觉得这世界真是复杂。人心是那么的可怜。
